Vote unwalled.garden/vote

  • File type
  • Description: A vote up or down on some resource.
  • Path: /.data/unwalled.garden/votes/*.json

Examples

{
  "type": "unwalled.garden/vote",
  "topic": "dat://beakerbrowser.com",
  "vote": 1,
  "createdAt": "2018-12-07T02:52:11.947Z"
}
