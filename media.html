Media unwalled.garden/media

  • File type
  • Description: A published item of content.
  • Path: /.data/unwalled.garden/medias/*.json

Notes

Media records are a way for sites to publish content. They should not be used for internal housekeeping. For instance, if a user embeds a gif into their post, the app should not also create a media record for the gif.

Media is defined by a subtype URL. Unwalled garden includes a set of useful subtypes. Applications are encouraged to use extensions to add custom information about the media.

Examples

{
  "type": "unwalled.garden/media",
  "subtype": "unwalled.garden/media#gif",
  "href": "dat://alice.com/images/angry-cat.gif",
  "title": "Bosco the angry cat",
  "tags": ["cats", "aww"],
  "createdAt": "2018-12-07T02:52:11.947Z"
}

Subtypes

Name URL File types
blogpost unwalled.garden/media#blogpost Web pages such as html or markdown.
document unwalled.garden/media#document Documents such as txt, pdf, docx, html, or markdown.
ebook unwalled.garden/media#ebook Ebooks such as txt, pdf, docx, html, or markdown.
file unwalled.garden/media#file Any.
gif unwalled.garden/media#gif Silent animated images or videos such as gif, webm, or mp4.
image unwalled.garden/media#image Images such as gif, png, or jpeg.
news-article unwalled.garden/media#news-article Web pages such as html or markdown.
page unwalled.garden/media#page Web pages such as html or markdown.
podcast-episode unwalled.garden/media#podcast-episode Audio such as webm, ogg, mp3, and wav.
song unwalled.garden/media#song Audio such as webm, ogg, mp3, and wav.
video unwalled.garden/media#video Videos such as webm and mp4.

Schema

{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "$id": "dat://unwalled.garden/media.json",
  "type": "object",
  "title": "media",
  "description": "A published item of content.",
  "required": [
    "type",
    "subtype",
    "href",
    "title",
    "createdAt"
  ],
  "properties": {
    "type": {
      "type": "string",
      "const": "unwalled.garden/media"
    },
    "subtype": {
      "type": "string"
    },
    "href": {
      "type": "string",
      "format": "uri"
    },
    "title": {
      "type": "string"
    },
    "description": {
      "type": "string"
    },
    "tags": {
      "type": "array",
      "items": {
        "type": "string",
        "maxLength": 100,
        "pattern": "^[A-Za-z][A-Za-z0-9-_?]*$"
      }
    },
    "createdAt": {
      "type": "string",
      "format": "date-time"
    },
    "updatedAt": {
      "type": "string",
      "format": "date-time"
    },
    "ext": {
      "type": "object"
    }
  }
}
