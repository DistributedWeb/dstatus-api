Library API

An API for dats that have been saved locally and/or published on the network.

import { library } from 'dat://unwalled.garden/index.js'

// read
await library.list({
  authors,
  types,
  keys,
  isSaved,
  isHosting,
  visibility,
  forkOf,
  isOwner,
  sortBy,
  offset,
  limit,
  reverse
})

// write
await library.configure(key, {isSaved, isHosting, visibility})

LibraryDat

The values returned by library functions will fit the following object shape:

Attribute Type Usage
key string The key of the dat
url string The url of the dat
author Object The dat that published the dat record (not necessarily the author of the dat)
 url string
 title string
 description string
 type string
 isOwner boolean Is the local user the owner of the author dat?
meta Object Information about the dat
 title string
 description string
 type string
 mtime number The timestamp of the last observed change to the dat
 size number The local size of the dat in bytes
 author url The url of the author of this dat
 forkOf url The url of the dat this dat is forked from
 isOwner boolean Is the local user the owner of the dat?
isSaved boolean Has the local user saved the dat?
savedAt number The timestamp of when the dat was saved to the local user’s library
isHosting boolean Is the local user hosting the dat?
visibility string The visibility of the dat and its record

Notes

The canonical record of a saved dat is stored privately. If the visibility is “public” then a link-entry will also be added to the public dats record.

Thumbnails of published dats should be published at /.data/thumbs/ using the key as the filename.

list(opts)

List dats by that are saved locally or published by authors.

Param Type Default Usage
opts Object
 authors string|string[] Filter by author URLs
 types string|string[] Filter by dat types
 keys string|string[] Filter by dat keys
 isSaved boolean Filter by: Has the local user saved the dat?
 isHosting boolean Filter by: Is the local user hosting the dat?
 visibility string 'all' Filter by this visibility. See visibility
 forkOf string Filter by: Only forks of this URL
 isOwner boolean Filter by: Is the local user the owner of the dat?
 sortBy string 'topic' One of: 'topic'
 offset number 0
 limit number
 reverse boolean false
Returns
Promise<LibraryDat[]>

configure(key, opts)

Configure a dat in the local user’s library.

Param Type Default Usage
key string Dat key (required)
opts Object
 isSaved boolean Save the dat to the local user’s library?
 isHosting boolean Host the dat using the local user’s devices?
 visibility string One of: 'public', 'unlisted', 'private'
Returns
Promise<void>
